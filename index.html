Gemini API Key Setup

ResuTailor runs entirely in your browser. We don't host an AI model server to keep our operations 100% free and keep your data private. To use the AI features, you need a Gemini API Key.

How to get a FREE API Key:

  1. Open the Google AI Studio API keys page: aistudio.google.com/apikey
  2. Sign in with any Google account.
  3. Choose "Create API key" and follow the prompts.
  4. Copy the generated key and paste it below.

Google occasionally moves things around — if the page looks different, just look for a "Create API key" option.

This key is only stored inside your browser's local storage and is sent directly to Google APIs. We never see or store your key.
